# Erise-AI 后端模块任务拆解文档（V1）

## 1. 阶段划分
### 第一阶段：骨架搭建
- 统一响应与异常
- Spring Security + JWT
- MyBatis-Plus 配置
- Redis 接入
- OpenAPI/Knife4j
- TraceId 日志

### 第二阶段：核心业务闭环
- 用户与登录
- 项目 CRUD
- 文件上传/下载/预览元数据
- 文档创建/保存/查询
- 标签管理
- 基础搜索

### 第三阶段：知识化与 AI 对接
- 文件解析任务
- 知识块抽取
- 索引状态管理
- AI 会话数据结构
- 内部业务 API 对接 AI 微服务

### 第四阶段：后台与增强
- 审计日志
- 任务监控
- 管理后台接口
- WebSocket
- 权限与安全增强

## 2. 模块任务
### auth
- 注册 / 登录 / refresh / logout / captcha
- 密码哈希
- token 签发
- 登录日志

### user
- 当前用户信息
- 用户资料更新
- 修改密码
- 偏好设置

### project
- 项目创建、查询、更新、归档、删除
- 项目概览统计
- 归属权限校验

### file
- 初始化上传 / 普通上传 / 分片上传
- MinIO/S3 封装
- 文件详情 / 下载
- 标签绑定
- 解析任务投递

### document
- 创建文档
- 保存草稿
- 发布文档
- 文档版本
- 文本快照抽取

### tag
- 标签 CRUD
- 文件/文档标签绑定

### search
- 关键词搜索
- 文件名搜索
- 文档内容搜索
- 搜索历史

### task
- 文件解析任务
- OCR/文本抽取/预览/索引
- 状态机与重试

### history
- 最近访问
- 最近编辑
- 最近下载
- 搜索历史

### admin
- 用户管理
- 文件审计
- 任务监控
- 审计日志
- 模型配置

### editor
- WebSocket
- 自动保存状态通知
- 任务通知

## 3. 重要约束
- Controller 轻
- Service 重
- Mapper 纯访问
- Entity 不直出
- AI 微服务不能绕过主体后端直接走主业务逻辑
